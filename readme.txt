Steps to run backend code

Step 1: Clone the github Repo

Step 2: Go to directory where you cloned the repo and get inside the repo directory

Step 3: Create a .env using your OpenAI API KEY, run: echo "OPENAI_API_KEY=YOUR_KEY" >> .env

Step 4: Install all the required python packages, run in your terminal: pip install -r requirements.txt

Step5: Run backend.py (python3 backend.py)
